Chicken fight monsters

Fight monsters and save the Chicken!

About game

This is a game for children 4-6 years old. Here you will find several types of tasks: mathematical, linguistic, listening and others. The child chooses the type of spell, gets the question, and if answer is correct - spell applies. And Chicken gets closer to winning!

Game rules

  1. The player chooses the spell first (attack ot heal).
  2. The player proceeds to the task screen. If answer is correct - spell applies.
  3. The monster applies a spell after player's mistake.
  4. If the player wins - the battle with the next monster. If you lose - shows a table of records, with the number of defeated monsters and the name of the hero.
